# Módulos — Bellex

> Última atualização: 2026-06-12

---

## App da Clínica

| Módulo | O que faz |
|--------|-----------|
| **Dashboard** | Visão geral do negócio — KPIs, faturamento, agenda do dia |
| **Agenda** | Calendário completo, bloqueios, encaixes, confirmação automática |
| **Clientes** | Cadastro completo, histórico, documentos, prontuário, consentimentos |
| **Cobranças** | Lançamentos, controle de pagamentos, Pix e cartão |
| **Faturamento** | Relatório financeiro, DRE simplificado, fluxo de caixa |
| **Produtos** | Catálogo com fotos, estoque, custo e preço de venda |
| **Serviços** | Serviços com duração, valor e especialistas habilitados |
| **Equipe** | Cadastro de especialistas, roles e permissões por função |
| **Marketing** | Segmentação de clientes + campanhas automáticas via WhatsApp e e-mail |
| **Pipeline (CRM)** | Funil de vendas e acompanhamento de leads |
| **↳ IA de Atendimento** | Agente IA que atende, tira dúvidas e agenda pelo WhatsApp automaticamente |
| **Mensagens** | Central de atendimento manual via WhatsApp integrado |
| **Relatórios** | Relatórios de agenda, faturamento, clientes e desempenho |
| **Agendamento online** | Link público para clientes agendarem sem precisar ligar |
| **Configurações** | Horários, integrações, notificações, whitelabel |

## Painel Workspace

| Módulo | O que faz |
|--------|-----------|
| **Dashboard** | KPIs consolidados de toda a rede — receita, clínicas ativas, uso |
| **Clínicas** | Cadastro, configuração e acesso a cada clínica individualmente |
| **Clientes** | Visão global de todos os clientes da rede |
| **Planos** | Definição dos planos que você oferece para suas clínicas |
| **Financeiro** | Receitas e cobranças da rede |
| **Licenças** | Controle de seats ativos, expiração e limites por clínica |
| **Usuários** | Gestão de usuários em toda a rede |
| **Relatórios** | Relatórios consolidados por clínica ou geral |
| **Notificações** | Alertas de uso, vencimentos e eventos das clínicas |
| **Suporte** | Central de tickets e feedbacks das clínicas |
| **Configurações** | Whitelabel, integrações (Resend, WhatsApp, IA), domínio |
